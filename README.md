# HR Analytics Dashboard (Power BI) 📊

### Project Overview
This project provides an end-to-end analysis of a company's workforce to understand **Employee Attrition** and **Salary Distribution**. By analyzing department performance and tenure, I identified the key factors contributing to employee turnover and provided actionable insights for HR leadership.

### Tech Stack
* **Visualization:** Power BI
* **Data Transformation:** Power Query (DAX for Custom Metrics)
* **Analysis:** Trend Analysis & Demographic Segmentation

### Key Insights
* **Attrition Hotspots:** The Sales department has the highest attrition rate (22%), primarily among employees with 1-2 years of tenure.
* **Salary Correlation:** Employees in the bottom 25% salary bracket are 3x more likely to leave the company compared to the top 25%.
* **Experience Impact:** New hires (0-1 year) represent the highest "Early Leaver" risk, suggesting a need for better onboarding.
